# Channel Skills (Ad Platform Skills)

This folder contains skills for ad platforms and marketing channels.

## How to add a new skill

1. Create a new `.md` file in this folder
2. Name it descriptively: `google-ads-bidding.md`, `meta-ads-audiences.md`, etc.
3. Start the file with a clear heading describing what the skill is about
4. Commit — Ema auto-discovers new files, no code changes needed

## Naming Convention

- `google-ads-*.md` — Google Ads skills
- `meta-ads-*.md` — Meta/Facebook Ads skills (future)
- `shopify-*.md` — Shopify skills (future)
- `tiktok-ads-*.md` — TikTok Ads skills (future)

## Current Skills

All existing Google Ads skills are in the parent `/skills` folder and will work from there (backward compatible). New skills should be added here.
